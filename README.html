php-logs-reader

A web interface for PSR-3 logging

Composer

Installation

From the Command Line:

composer require alexivashchenko/php-logs-reader:dev-master

Update

composer update alexivashchenko/php-logs-reader

Basic Usage

Create logs

<?php

require 'vendor' . DIRECTORY_SEPARATOR . 'autoload.php';

$LogsGenerator = new Logs\Generator([
    'path' => 'logs' . DIRECTORY_SEPARATOR,
    'channel' => 'dummy',
    'log_files_format' => 'log',
    'line_type' => 'string',
]);

$LogsGenerator->info('New user added', [
    'id' => 1,
    'firstname' => 'Barack',
    'lastname' => 'Obama',
    'status' => 'ex-president',
]);
$LogsGenerator->error('Error adding user');
